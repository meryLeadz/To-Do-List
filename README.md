# To-Do List App

## Description

This is a simple To-Do List application built with HTML, CSS, and JavaScript. It allows users to add, delete, and mark tasks as completed. The tasks are stored in the browserâ€™s local storage, ensuring they persist even after a page reload.

## Features

- Mark tasks as completed
- Delete tasks from the list

- **script.js**: The JavaScript functionality for adding, marking, and deleting tasks.
